# JuicyScore

## Enoncé
Un critique culinaire souhaite avoir une application mobile afin de lui permettre de noter les
restaurants qu’il essaye tout au long de l’année.

À minima il veut pouvoir saisir:

- le nom du restaurant
- la date et l’heure de son repas
- une note pour la décoration
- une note pour la nourriture
- une note pour le service
- une zone de description / critique

N’ayant pas toujours de réseau, ces informations devront être stockées dans une base
de données directement sur le téléphone.
